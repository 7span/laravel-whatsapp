# Why Laravel WhatsApp?

Laravel WhatsApp is a powerful package that integrates WhatsApp Cloud API with your Laravel applications, making it easy to send template messages through WhatsApp Business API.

## Key Features

### 1. Simple Integration
- Easy installation through Composer
- Automatic service provider and facade registration
- Straightforward configuration through environment variables

### 2. Template Message Support
- Send pre-approved WhatsApp templates
- Support for dynamic content insertion
- Handle multiple template parameters


### 3. Flexible Configuration
- Configurable API endpoints
- Easy token management
- Custom separator support for message parameters
- Environment-based configuration
